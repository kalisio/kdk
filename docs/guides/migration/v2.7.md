# v2.7 - March 2026

More details can be found in the related [milestone](https://github.com/kalisio/kdk/milestone/15) on GitHub.

## Major breaking changes

None yet reported.

## Major new features

👉 New configuration service utilities used to store configuration options that can be updated by the application from the client.

👉 New boolean `focus` field property to be used in your schema in order to set the default focused field.

👉 New boolean `expand` field property to be used in your stickies in order to fit the page width.
